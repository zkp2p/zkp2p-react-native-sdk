# zkp2p-react-native-sdk

React Native SDK for ZKP2P

## Installation

```sh
yarn add @zkp2p/zkp2p-react-native-sdk
```

## Usage


```js
import { Zkp2pProvider, useZkp2p } from 'zkp2p-react-native-sdk';
```
